# LOD Architecture (Layer 1)

The world is a recursive tree with branching factor 3. Every node has 3x3x3 = 27 children. Terminal nodes are block types (stone, grass, air). Non-terminal nodes are compositions of children. There is no special "leaf" layer -- every layer is structurally identical, and every layer is "just Minecraft" where the blocks happen to be meshes from the layer below. Content is authored bottom-up as "canned structures": you design 200 trees, arrange them into 50 forests, arrange forests into biomes, biomes into planets. Each cell in the player's 27x27x27 view grid contains a pre-baked mesh that preserves 3 layers of visual detail below the cell level, so a tree that occupies one cell still looks like a tree (its 27x27x27 block structure is visible in the mesh). The tree only becomes a single pixel 6 zoom levels above its native layer. Base 3 was chosen because it is the smallest odd branching factor -- odd means every node has a center cell (1,1,1), which matters because natural structures have spines (trunks, doorways, torsos) and because accessing the center of an even-base node requires touching all 8 children at the junction.
